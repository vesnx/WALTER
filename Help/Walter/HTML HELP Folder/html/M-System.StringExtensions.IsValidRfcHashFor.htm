Parameters

base64String
Type: String
The base64-encoded hash string to validate against. Typically, this is the stored hash value in a database.
plainText
Type: String
The plain-text string to compare. For instance, a password input by a user.

Return Value

true if the plain-text string, when hashed, matches the provided hash; otherwise, false.

Remarks

This method is particularly useful for verifying user passwords or other sensitive data where the original value is hashed for security. For example, during user authentication, the method can compare the hash of the entered password against the stored hash to confirm identity.

Examples

This example demonstrates how to use IsValidRfcHashFor in a user authentication scenario:
C#
// Assume 'storedHash' is the previously stored hash of the user's password
   string storedHash = RetrieveStoredPasswordHash(username);

   // 'userInput' is the password entered by the user during the login process
   string userInput = GetUserInputPassword();

   // Check if the hash of the user's input matches the stored hash
   bool isValidLogin = storedHash.IsValidRfcHashFor(userInput);

   if (isValidLogin)
   {
       // Authentication successful
       Console.WriteLine("Login successful.");
   }
   else
   {
       // Authentication failed
       Console.WriteLine("Invalid username or password.");
   }
Simple round-trip sample
string secret = Guid.NewGuid().ToString();
string password = "Password";

// Hash the secret
var hash = secret.RfcHash(password, 10000);

// Now verify if the original secret, when hashed, matches the hash
// The first parameter of IsValidRfcHashFor should be the hash,
// and the second parameter should be the secret (plain text)
Assert.IsTrue(hash.IsValidRfcHashFor(password));
