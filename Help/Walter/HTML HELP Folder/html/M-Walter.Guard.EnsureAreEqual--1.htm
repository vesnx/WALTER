Remarks

This method logs an error and throws an ArgumentsAreNotEqualException if the expected and actual values are not equal. It utilizes platform-specific logging (Event Log on Windows, syslog on Linux) when no external logging framework is configured. If either expected or actual implements [IEqualityComparer{T}], that comparer is used; otherwise, the default equality comparer for type T is used.

Examples

C#
int expectedValue = 5;
int actualValue = CalculateValue();

// Throws ArgumentsAreNotEqualException if actualValue is not 5, with the expected and actual parameter names automatically captured in supported .NET versions.
